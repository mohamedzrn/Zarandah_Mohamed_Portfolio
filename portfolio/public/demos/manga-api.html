Architecture

How It Works

Three-layer architecture that keeps things clean, fast, and scalable.

01

Browse

Users search and filter manga by title, genre, or author through the Lumen JS frontend interface.

02

Fetch

The PHP REST API queries the MySQL database and returns optimized JSON responses in milliseconds.

03

Display

Data is rendered into dynamic, responsive manga cards with cover art, ratings, and detail views.
